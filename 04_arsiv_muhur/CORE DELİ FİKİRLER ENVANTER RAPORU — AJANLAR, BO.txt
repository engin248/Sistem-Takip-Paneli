## 1.2 AI Provider Durumları (aiProvider.ts — 427 satır'dan doğrulanmış)

### SAĞLAYICI 1: OLLAMA (YEREL)
- **Dosya:** `src/lib/aiProvider.ts` satır 112-174
- **URL:** `http://localhost:11434/api/chat`
- **Model:** `llama3:latest` (env'den değiştirilebilir: OLLAMA_MODEL)
- **Timeout:** 15 saniye (env: AI_TIMEOUT_MS)
- **Maliyet:** 0₺
- **Durum:** KOD TAM, çalışması Ollama servisinin açık olmasına bağlı
- **Health Check:** `GET http://localhost:11434/api/tags` (30sn cache)
- **Circuit Breaker:** Aktif — hata eşiği aşılırsa otomatik durdurur
- **Özellikler:**
  - JSON mode desteği (format: 'json')
  - temperature ayarı (varsayılan: 0.2)
  - max_tokens ayarı (varsayılan: 200)
  - Stream: false (tam yanıt bekler)
  - Token sayımı (eval_count + prompt_eval_count)

### SAĞLAYICI 2: OPENAI (DIŞ)
- **Dosya:** `src/lib/aiProvider.ts` satır 176-237
- **URL:** `https://api.openai.com/v1/chat/completions`
- **Model:** `gpt-4o-mini` (env'den değiştirilebilir: OPENAI_MODEL)
- **API Key:** env: OPENAI_API_KEY
- **Maliyet:** ~$0.002/istek
- **Durum:** KOD TAM ama ZORLA DEVRE DIŞI
- **Devre dışı bırakma:** `FORCE_DISABLE_OPENAI` env varsayılan olarak `true`
  - `aiProvider.ts` satır 67: `forceDisableOpenAI: process.env.FORCE_DISABLE_OPENAI !== 'false'`
  - Yani açıkça `false` yazılmadıkça OpenAI HER ZAMAN KAPALI
- **Özellikler:**
  - JSON mode desteği (response_format: json_object)
  - Bearer token auth
  - Token sayımı (usage.total_tokens)

### SAĞLAYICI 3: LOCAL (KURAL TABANLI)
- **Dosya:** Çeşitli (her modül kendi fallback'ini tanımlar)
- **URL:** Yok — internet gerektirmez
- **Model:** Yok — deterministik kural mantığı
- **Maliyet:** 0₺
- **Durum:** HER ZAMAN AKTİF
- **Nasıl çalışır:** AI (Ollama veya OpenAI) erişilemezse, `aiComplete()` fonksiyonu `null` döner. Çağıran fonksiyon (analysisEngine, agentWorker vs.) kendi yerel kurallarını çalıştırır.

### AI ÇAĞRI ZİNCİRİ (Fallback sırası)
```
1. Ollama çalışıyor mu? → EVET → Ollama'ya gönder (0₺)
                          HAYIR ↓
2. OpenAI aktif mi?     → FORCE_DISABLE_OPENAI=true → ATLA
                          HAYIR ↓
3. null dön → Çağıran fonksiyon kendi kuralını çalıştırır (0₺)
```
